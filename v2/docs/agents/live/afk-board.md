# AFK Live Agent Board

This directory provides the human-readable views of the AFK Live Agent Board, which tracks the status of parallel autonomous agents operating on this repository.

## The Board

Maintainers can view the dashboard by opening the static HTML page:

[View HTML Dashboard](./html/index.html)

### Architecture and Source of Truth

The authoritative source of truth for agent states is the JSON state artifact generated by the issue/PR lifecycle:

`governance/agents/live/afk-runs.json` (or another designated path like `docs/agents/live/html/afk-runs.json`).

The HTML and Markdown files in this directory **must not** become the source of truth. They are merely human-readable generated views.

```text
governance/agents/live/afk-runs.json
  -> docs/agents/live/afk-board.md
  -> docs/agents/live/html/index.html
```

### Refreshing the Data

The HTML dashboard uses static JavaScript to fetch `afk-runs.json` periodically (every 60 seconds). It operates entirely on the client-side without external dependencies (no npm, no WebSockets, no external CDNs) to ensure it works directly from a local file checkout or basic static hosting.

### Generation

The workflow for updating this board is intended to be:

1. A watchdog/update state job runs in CI or via another mechanism.
2. It generates or updates the `afk-runs.json` file.
3. This triggers a copy/normalization of the data into the `docs/agents/live/html/` folder.
