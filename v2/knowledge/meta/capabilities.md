# What TARS Can Do

## Language Model Integration
- Connect to Ollama, OpenAI, vLLM, Anthropic, Google Gemini
- Route requests based on model hints (code, reasoning, fast)
- Stream responses token-by-token
- Manage token budgets

## Tool Calling
- Parse multiple tool call formats (legacy, JSON, XML-style)
- Execute tools and inject results back into conversation
- Discover tools via reflection

## Memory & Knowledge
- Maintain conversation history
- Store persistent knowledge in markdown wiki
- Semantic search via vector embeddings

## Agentic Patterns
- Chain of Thought reasoning
- ReAct (Reason + Act) loops
- Plan and Execute decomposition

## Self-Improvement
- Curriculum-based learning
- Track successful/failed approaches
- Record learnings to knowledge base
