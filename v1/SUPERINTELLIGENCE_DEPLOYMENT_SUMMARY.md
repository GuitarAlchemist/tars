## 🚀 DEPLOYMENT COMMANDS

### Quick Start
```powershell
# Test superintelligence capabilities
powershell -ExecutionPolicy Bypass -File test-superintelligence-docker.ps1 -Action test

# Deploy full stack
powershell -ExecutionPolicy Bypass -File test-superintelligence-docker.ps1 -Action deploy

# Check status
powershell -ExecutionPolicy Bypass -File test-superintelligence-docker.ps1 -Action status
```

### Blue/Green Operations
```powershell
# Deploy to blue environment
.\scripts\blue-green-deploy.ps1 -Action deploy -TargetColor blue

# Deploy to green environment  
.\scripts\blue-green-deploy.ps1 -Action deploy -TargetColor green

# Switch traffic to green
.\scripts\blue-green-deploy.ps1 -Action switch -TargetColor green

# Rollback to previous environment
.\scripts\blue-green-deploy.ps1 -Action rollback

# Check deployment status
.\scripts\blue-green-deploy.ps1 -Action status
```

### Docker Compose Operations
```bash
# Start full superintelligence stack
docker-compose -f docker-compose.superintelligence.yml up -d

# Scale blue environment
docker-compose -f docker-compose.superintelligence.yml up -d --scale tars-superintelligence-blue=2

# View logs
docker-compose -f docker-compose.superintelligence.yml logs -f tars-superintelligence-blue

# Stop stack
docker-compose -f docker-compose.superintelligence.yml down
```
