## 🔧 **Configuration**

### **Evolution Settings** (`tars.evolution.*`)
```json
{
  "tars": {
    "evolution": {
      "enableSelfModification": true,
      "maxModificationsPerCycle": 3,
      "minConfidenceThreshold": 0.8,
      "maxRiskLevel": "Medium",
      "intervalMinutes": 60,
      "enableAutonomous": true,
      "requireApproval": false,
      "backupBeforeModification": true,
      "enableRollback": true,
      "maxDepth": 5
    }
  }
}
```

### **Safety Controls**
- **Confidence Threshold**: Minimum confidence required for modifications (default: 80%)
- **Risk Level Limits**: Maximum acceptable risk level (Low/Medium/High)
- **Human Approval**: Optional requirement for human approval
- **Backup System**: Automatic backup before any modification
- **Rollback Capability**: Automatic reversal of failed changes
