# Plan TARS Enhancement

Plan a new TARS enhancement or autonomous capability improvement using Agent OS methodology.

Refer to the instructions located in this file:
@.agent-os/instructions/core/plan-product.md

## Usage

Use this command when you want to:
- Plan a new TARS autonomous capability
- Enhance existing TARS functionality
- Integrate new features with TARS metascript system
- Improve TARS performance or self-improvement capabilities

## Expected Outputs

This command will create:
- `.agent-os/product/tars-mission.md` - Enhancement vision and objectives
- `.agent-os/product/tars-tech-stack.md` - Technical implementation details
- `.agent-os/product/tars-roadmap.md` - Development phases and tasks
- `.agent-os/product/autonomous-objectives.md` - Autonomous agent goals

## TARS-Specific Considerations

- Focus on autonomous capabilities and self-improvement
- Ensure CUDA acceleration and real GPU performance
- Maintain zero tolerance for simulations/placeholders
- Follow FLUX metascript integration patterns
- Target 184M+ searches/second performance where applicable
